Pelican Comment System

Pelican Comment System is a plugin for pelican and allows you to add static comments to your articles and pages.

Comments are stored in files in formats that can be processed by Pelican (e.g., Markdown and reStructuredText). Each comment resides in its own file.

Features

  • Static comments for each article/page
  • Replies to comments
  • Avatars and Identicons
  • Comment Atom feed for each article/page
  • Easy styleable via themes
  • Comments can be closed or hidden for each article/page separately
  • Supports:
    • Python: 2.7 and 3 (3.3 - 3.6)
    • Pelican: 3.4 and newer
